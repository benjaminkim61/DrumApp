# Metronome Click Sounds

Place your metronome sound files in this folder with the following names:

## Style 1 (Primary)
- `click1.mp3` - Normal click sound for Style 1
- `click_accent1.mp3` - Accented click sound for Style 1 (played on downbeats)

## Style 2 (Secondary) 
- `click2.mp3` - Normal click sound for Style 2
- `click_accent2.mp3` - Accented click sound for Style 2 (played on downbeats)

## Notes:
- All files should be MP3 format
- Keep file sizes small (< 100KB each) for optimal performance
- Subdivision sounds are automatically generated by pitch-shifting the normal click sounds down
- The app will automatically use the appropriate click style based on user settings

## Recommended Sound Characteristics:
- Short duration (50-100ms)
- Clear attack
- Minimal sustain/decay
- No reverb/echo for best timing accuracy
